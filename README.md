# lab-connect
O LabConnect é uma rede de conexão acadêmica onde o professor publica a vaga (bolsa ou voluntariado) e o aluno mantém um perfil atualizado com suas competências técnicas e áreas de interesse. A solução conta com um módulo de inteligência para sugerir os alunos mais aptos para cada vaga, automatizando o "match" que hoje é feito manualmente.
